<div> <h2><a id="hints-to-find-what-you-are-looking-for" class="anchor" href="#hints-to-find-what-you-are-looking-for">¶</a>Hints to find what you are looking for</h2><p>Emacs is self documented, the code's documentation is searchable, accessible within emacs. </p> <ul><li>functions, commands, variables are usually prefixed by the mode name. For example, org-roam is the prefix for <strong>org-roam</strong>-db-build-cache command.</li> <li>All these shortcuts present you with a minibuffer, or a selection prompt (ex: ivy, helm). The user experience really depends on your configuration.</li> </ul> <h2><a id="cheatsheet" class="anchor" href="#cheatsheet">¶</a>Cheatsheet</h2><p>Here are the most helpful shortcuts to reach the Emacs help system: </p> <table class="table table-dark"><thead class="thead-dark"><tr><td>Key</td><td>command</td><td>Purpose</td></tr></thead> <tbody><tr><td>C-h i</td><td>info</td><td>User guide for emacs and important modes</td></tr> <tr><td></td><td></td><td>Also displays Linux tools <a href="https://en.wikipedia.org/wiki/Info_(Unix)">info pages</a></td></tr> <tr><td>C-h r</td><td>info-emacs-manual</td><td>Emacs User Manual</td></tr> <tr><td>C-h f</td><td>describe-function</td><td>show detailed information about functions and commands</td></tr> <tr><td>C-h v</td><td>describe-variable</td><td>variables detailed information</td></tr> <tr><td>C-h b</td><td>describe-bindings</td><td>describes the enabled key bindings with the functions/commands they call</td></tr> <tr><td>C-h m</td><td>describe-mode</td><td>describes the current major mode enabled with all enabled minor modes</td></tr> <tr><td></td><td></td><td>Also lists key bindings</td></tr> <tr><td>C-h a</td><td>apropos-command</td><td>Search documentation related to a command</td></tr> <tr><td>C-h d</td><td>apropos-documentaion</td><td>Search documentation for a specific keyword</td></tr></tbody> </table> </div>

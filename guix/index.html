GNU Guix

<div><p><em>A liberating, dependable, and hackable functional OS and package manager.</em> </p> <h2><a id="welcome-to-the-system-crafters-community-gnu-guix-wiki" class="anchor" href="#welcome-to-the-system-crafters-community-gnu-guix-wiki">¶</a>Welcome to the System Crafters' Community GNU Guix Wiki!</h2><p>GNU Guix is a functional, reproducible GNU/Linux package manager and OS. A free (as in freedom), Guile Scheme based alternative to Nix, Guix provides the power of a functional OS and more. While powerful, many of Guix's configurations and options seem arcane and inaccessible. This wiki is here to show that the complete opposite is true! </p> <p>If you enjoy video content, you might want to watch the System Crafters playlist <a href="https://www.youtube.com/watch?v=iBaqOK75cho&list=PLEoMzSkcN8oNxnj7jm5V2ZcGc52002pQU">Craft Your System with GNU Guix</a>. </p> <h3><a id="why-if-so-many-other-places-and-the-manual-exist" class="anchor" href="#why-if-so-many-other-places-and-the-manual-exist">¶</a>Why if so many other places (and the manual) exist?</h3><p>Truthfully, there aren’t that many other places. The System Crafters' wiki is attempting to enable more people to use GNU Guix with free and non-free software. The community should understand and be open to the fact that not all can realistically use only free software; this is not to say that the FSF is wrong or bad, but that we should ease people into using free as in freedom software and utilities. The System Crafters community believes there is a lot we can do to improve the experience for all. </p> <h3><a id="where-to-start" class="anchor" href="#where-to-start">¶</a>Where to start?</h3><h4><a id="if-youve-never-used-or-heard-of-gnu-guix" class="anchor" href="#if-youve-never-used-or-heard-of-gnu-guix">¶</a>If you've never used or heard of GNU Guix</h4><p>you should start at <a href="/guix/faqs">FAQs</a>. </p> <h4><a id="if-youve-used-nix-before-or-are-feeling-courageous" class="anchor" href="#if-youve-used-nix-before-or-are-feeling-courageous">¶</a>If you've used Nix before, or are feeling courageous</h4><p>start right away at <a href="https://guix.gnu.org/manual/en/html_node/System-Installation.html">Official Installation Guide</a> if you want a fully free experience, or if you are not sure what that means or want nonfree (recommended choice), start at the <a href="/guix/nonguix-installation-guide">Nonguix Installation Guide</a>. </p> <p>Note that the official installation guide is from the GNU Guix manual, and links to that site. It is still recommended to read <a href="/guix/general-recommendations">general recommendations</a> afterwards, although you might not want to follow the nonfree advice on there. </p> <h3><a id="featured-news-and-articles" class="anchor" href="#featured-news-and-articles">¶</a>Featured News and Articles</h3><h4><a id="possibly-randomly-select-articles-here-or-show-commonly" class="anchor" href="#possibly-randomly-select-articles-here-or-show-commonly">¶</a>Possibly randomly select articles here, or show commonly</h4><p>sought after things (Firefox, Nix.) </p> </div>
